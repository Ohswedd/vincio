# Vincio two-level public surface.
# Every public subpackage's __all__, one symbol per line, sorted by
# (subpackage, symbol), as: <subpackage> <tag> <symbol>. The tag classifies
# the symbol against the frozen top-level surface (docs/reference/public-
# surface.txt): TOP = re-exported in vincio.__all__ (the same object);
# DUP = the name also exists at the top level but as a different object (an
# intentional collision, e.g. the tabular data.Dataset beside the eval
# Dataset) so the subpackage symbol is reachable only by deep import;
# SUB = subpackage-only public. Re-freeze deliberately: regenerate with
# `python -m vincio._surface --freeze` and review the diff. Guarded by
# tests/test_surface_consistency.py.
